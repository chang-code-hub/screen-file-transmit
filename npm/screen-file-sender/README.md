# screen-file-sender

Screen file sender - transmit files via DataMatrix codes displayed on screen.

## Installation

```bash
npm install -g @chang-code-hub/screen-file-sender
```

## Usage

```bash
# Launch the sender GUI
screen-file-sender

# Or use the short alias
sfs
```

## Requirements

- Windows OS
- .NET Framework 4.6.1 or later

## CLI

This package wraps the Windows desktop application. Running the command launches the WPF GUI.
